Problema que soluciona:
    Seguridad ciudadana para la poblacion en general
    Validacion de automoviles sospechosos (taxis y otros) para evitar secuestros   
    Recopilicacion de datos de seguridad para los creadores de politicas


Solucion planteada:

Una aplicacion movil y web con las siguientes funcionalidades.
    Reconocimiento de placas para validacion si un auto es el modelo registrado o si es robado
    Mapa de crimenes y eventos que afecten a la seguridad
    Posibilidad de subir estos datos mediante:
        La aplicacion movil
        La aplicacion web
        Twitter
        SMS
